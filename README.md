# Water Quality Monitor

This project monitors water quality and provides a platform for users (citizens, NGOs, authorities, admins) to register, log in, and view water quality data on a map.

## Tech Stack
- Backend: FastAPI (Python)
- Database: PostgreSQL
- Authentication: JWT
- Frontend: React.js + Tailwind CSS
- Maps: Leaflet or Mapbox

## Milestone 1 Scope
### Backend (Weeks 1–2)
- FastAPI project structure (routers, models, schemas, services, database config)
- PostgreSQL schema for Users
- SQLAlchemy models
- Database connection
- Alembic migrations (optional)
- REST APIs: POST /auth/register, POST /auth/login
- Password hashing, JWT token, role-based user support

### Frontend (Weeks 1–2)
- React project with Tailwind CSS
- Pages: Login, Register, Dashboard (protected)
- JWT-based authentication, API integration, token storage, logout
- Dashboard: navbar, sidebar placeholder, base map view (India)
